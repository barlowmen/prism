# Section: Tailoring playbook by archetype

## Canonical heading

`## Tailoring playbook by archetype`

## What this section captures

For each target archetype (from **Career objectives**), the specific tailoring rules the drafting agent should follow: length, file format, PhD placement, title handling, summary voice, what to highlight, what to cut.

This section turns archetype-level intent into per-resume rules.

## Questions to cover

For **each priority archetype** from Career objectives, ask:

1. **Length.** 1 page or 2? Conditional? (e.g., "1 page if dense, else 2" for frontier labs.)
2. **File format.** PDF or DOCX? Which ATS dominates in that archetype's hiring? (Workday → DOCX; Greenhouse → either; LinkedIn → keep in sync separately.)
3. **PhD placement** (if applicable). Top of resume? Inside Education? Mentioned in summary as forward-looking?
4. **Title handling.** Lead with HR title? Functional title? Parenthetical scope-clarifier? Pull-quoted summary line that names the scope?
5. **Summary voice.** What does the summary open with — thesis, scope numbers, named systems, forward-looking hook? In what order?
6. **What to drop entirely** for this archetype (FedRAMP language? Older roles? Most certs?).
7. **What to highlight.** Specific accomplishments framed in the archetype's vocabulary (eval program → "AI governance and risk management at scale" for enterprise; → "high-stakes-domain trust" for frontier lab).
8. **Mission/voice notes.** Anything specific to that archetype's hiring culture? (E.g., frontier labs prefer authentic statements over fake-alignment-researcher framing.)
9. **Any explicit go/no-go gate** before drafting for that archetype? (E.g., PM roles → confirm the user actually wants this before letting the agent draft a PM-shaped resume.)

## Hard rule for this section

Each archetype block should be **dense with imperatives** the drafting agent can follow without re-deriving the strategy. If a rule is vague ("emphasize impact"), ask a follow-up until it's specific ("lead the summary with budget + headcount + named programs, in that order").
